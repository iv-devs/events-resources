Pensando en Objetos
  1. El Objeto ES ... --> PROPIEDAD Simple
  2. El Objeto TIENE ...
    • 1 Objeto : PROPIEDAD Compuesta
    • +1 Objeto : PROPIEDAD Colectora

  3. El Objeto PUEDE ... (verbo)
    • El objeto PUEDE volar -->Metodo()
    • El objeto PUEDE flotar -->Metodo()
    • El objeto PUEDE correr -->Metodo()

foreach($config as $seccion=>$array) {
    foreach($array as $constante=>$valor) {
        define($constante, $valor);
    }
}
if(!PRODUCTION) {
